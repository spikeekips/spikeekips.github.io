import gzip
import StringIO

def encode_zlib (s, ) :
    return s.encode('zlib', )

def encode_gzip (s, ) :
    _s = StringIO.StringIO()
    with gzip.GzipFile(mode='wb', fileobj=_s, ) as f :
        f.write(s, )

    return _s.getvalue()

def decode_gzip (s, ) :
    _s = StringIO.StringIO(s, )
    with gzip.GzipFile(mode='r', fileobj=_s, ) as f :
        return f.read()

func_compress = lambda f, z='zlib' : lambda x : (encode_zlib if z == 'zlib' else encode_gzip)(str(f(x)))

Conclusion

I compared which serializing method and tool shows the best compaction result with or without compressing. As the result is shown above, I could not find any meaningful differences between the methods. At first, I have expected the bson will show the best compaction result because it can serialize as binaries, but on the contrary, bson makes the input to be oversized.
